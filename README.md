# fofaa_ides
这是一个基于fofa网络空间测绘平台制作的一款url链接爬虫。（软件是基于TK，requests,re等模块做的，需要安装requests模块）
需要输入自己的账号的cookie才能运行，否则只能输出一页，软件配备了页数的选择和语法的搜索。
软件毕竟才研发了一两天所以并不是非常的完善，线程数仅支持两个。
